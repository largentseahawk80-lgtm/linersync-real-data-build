# LinerSync QC - Real Data Build AutoFill

This version adds Constant Job Data so values that stay the same auto-fill every new log until changed.

## Auto-filled until changed
- Project / site
- Client / owner
- QC technician
- Active roll number
- Active panel / pond / area
- Liner type
- Liner thickness
- Liner width
- Texture / color
- Installer / crew
- QC / CQA company
- Wedge machine
- Extrusion welder
- Rod / resin lot
- Shift / weather
- GPS and 12-hour date/time on every capture

## Modules
- Roll Inventory
- Panel Placement
- Wedge Weld Test
- Extrusion Weld Test
- Air Test
- Destructive Test
- Repair / Patch / Bead
- Daily Log

## Exports
- CSV export includes the constant values on every record
- JSON backup/restore includes project, constants, logs, GPS, and photos

Upload these files to GitHub Pages. For GitHub Pages root folder issue, upload the files themselves into the repo root, not inside a folder.
