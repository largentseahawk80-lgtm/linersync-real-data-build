# LinerSync QC - Real Data Build

This is a phone-ready prototype/PWA for geosynthetic liner QC field logging.

## What works now
- Project setup
- Tap-to-capture records
- GPS + 12-hour timestamp
- Photo capture
- Last Logs screen
- Local phone/browser saving
- CSV export
- JSON backup and restore
- Offline PWA cache after first load

## Modules included
- Roll Inventory
- Panel Placement
- Wedge Weld Test
- Extrusion Weld Test
- Air Test
- Destructive Test
- Repair / Patch / Bead
- Daily Log

## How to use
Open `index.html` in a browser, or upload these files to a GitHub Pages repo / Netlify / Cloudflare Pages.

For phone install:
1. Open the live page in Chrome/Safari.
2. Tap browser menu.
3. Tap Add to Home Screen.

## Important
This version saves locally first. Export JSON backups often until cloud sync is added.
